The BusyBox source code and the patches can be downloaded here:
http://busybox.net/downloads/busybox-1.20.2.tar.bz2
http://busybox.net/downloads/fixes-1.20.2/busybox-1.20.2-kernel_ver.patch
http://busybox.net/downloads/fixes-1.20.2/busybox-1.20.2-ntpd.patch
http://busybox.net/downloads/fixes-1.20.2/busybox-1.20.2-pkg-config-selinux.patch
http://busybox.net/downloads/fixes-1.20.2/busybox-1.20.2-sys-resource.patch

The patch "ash-fix-read-s-ctrl-c.patch" is cherry-picked from the upstream 
git source tree. It fixes an annoying bug for me.

The BusyBox binary is compiled on Ubuntu 12.04.2 LTS (32-bit).

After downloading and unpacking the source code, copy the file "config" and 
the ".patch" files into the source code directory. Then apply the patches and 
the config:

 $ cd busybox-1.20.2
 $ patch -p 1 <busybox-1.20.2-kernel_ver.patch
 $ patch -p 1 <busybox-1.20.2-ntpd.patch
 $ patch -p 1 <busybox-1.20.2-pkg-config-selinux.patch
 $ patch -p 1 <busybox-1.20.2-sys-resource.patch
 $ patch -p 1 <ash-fix-read-s-ctrl-c.patch
 $ mv config .config

Finally, build BusyBox:
 $ make
 $ make install

The binary will be ready in a directory named "_install" inside the busybox 
source, as well as the necessary symlinks.
